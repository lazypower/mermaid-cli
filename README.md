# MERMAID CLI

### ... in a can

This whopper of an image bakes in mermaid-cli. A node application that is useful
in keeping diagrams as code.

## Example Usage

```shell
docker run --rm -ti --user $(id -u):$(id -g) -v $PWD:/host local/mermaid-cli -i dns.mmd -o dns.png
```
